How to trace in Photopea (free, browser-based)
▾
1

Open the tracing reference

Click "Open in Photopea" above. The faded image opens directly in Photopea in a new tab. If it doesn't open automatically, download it and open at photopea.com.

2

Add a new layer and set opacity

In Photopea: Layer → New → Layer. Name it DESIGN. Set the original image layer opacity to 40–50% so you can see through your tracing.

3

Trace the main body

Select the Pen tool (P). Trace the outer silhouette of each motif — body, head, tail. Use the Layer Map reference (white/cream areas) to guide you. Close each path and fill with solid black.

4

Trace the wing and detail areas

On a new layer, trace the lavender/purple areas from the Layer Map — these are the wing shading and feather patterns. Trace the gold areas separately if you want them as a second shuttle (Meena).

5

Export as black on white PNG

Hide the original image layer. Set background to white. File → Export As → PNG. This is your clean design file.

6

Upload to BMP Generator

Go to the BMP Generator tab and upload your traced PNG. Use 960 pins for best detail. The app will detect the design perfectly — no grid, no noise, clean output.

Tip: You don't need to trace every pixel perfectly. The Pen tool with smooth curves takes ~20–30 minutes for a single bird motif. Once you have one clean motif, you can duplicate it across the canvas for the repeat pattern. Use 960 pins in the BMP Generator so the wing detail renders at 3–4px minimum width.

How to trace in Photopea (free, browser-based)
▾
1

Open the tracing reference

Click ↓ Download on the Tracing Reference panel, then click Open in Photopea ↗. Photopea opens — drag the downloaded file straight onto the Photopea tab. Or use ⤢ Full size to see it at full resolution first.

2

Add a new layer

In Photopea: Layer → New → Layer. Name it DESIGN. Set the original image layer opacity to 40–50% so you can see through your tracing.

3

Trace each colour layer

Use the Pen tool (P) to trace each motif. Refer to the Layer Map to understand which areas belong to which design element. Close each path and fill with solid black.

4

Export as black on white PNG

Hide the original image layer. Set background to white. File → Export As → PNG. This is your clean design file.

5

Upload to BMP Generator

Go to the BMP Generator and upload your traced PNG. Use 960 pins for best detail. The app will detect the design cleanly — no grid, no noise.

Tip: If the Auto-Cleaned result already looks good — mostly the right shapes, few noise specks — click Send to Generator directly. Skip manual tracing unless the design has fine detail you need to preserve exactly. Try adjusting the Capture strength slider and hitting Re-process before deciding to trace manually.
